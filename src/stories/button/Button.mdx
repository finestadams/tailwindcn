# Button

The `Button` component is used to trigger actions or events. It supports multiple variants, sizes, and additional features like icons and full-width styling.

---

## Installation

To use the `Button` component in your project, after installing the package:
For configuration details, refer to the [Configuration Guide](?path=/docs/introduction--docs).

```bash
wind add button
```

---

## Usage

Import the `Button` component into your project:

```tsx
import { Button } from "@component/ui/button";
```

---

## Variants

### Primary Button

The primary button is used for the main action on a page.

```tsx
<Button variant="primary" size="md">
  Primary
</Button>
```

<Story name="Primary" of={ButtonStories.Primary} />

---

### Outline Button

The outline button is used for secondary actions.

```tsx
<Button variant="outline" size="md">
  Outline
</Button>
```

<Story name="Outline" of={ButtonStories.Outline} />

---

### Danger Button

The danger button is used for destructive actions, such as deleting data.

```tsx
<Button variant="danger" size="md">
  Danger
</Button>
```

<Story name="Danger" of={ButtonStories.Danger} />

---

### Pill Button

The pill button has rounded corners for a softer appearance.

```tsx
<Button variant="primary" size="md" pill>
  Pill Button
</Button>
```

<Story name="Pill" of={ButtonStories.Pill} />

---

### Full Width Button

The full-width button spans the entire width of its container.

```tsx
<Button variant="primary" size="md" fullWidth>
  Full Width Button
</Button>
```

<Story name="Full Width" of={ButtonStories.FullWidth} />

---

### Small Button

The small button is used when space is limited.

```tsx

```

### Medium Button

The medium button is the default size.

  Medium Button
</Button>

The large button is used for emphasis or when more space is available.

## Notes

- Use the `variant` prop to customize the button's appearance.
- Combine `pill` and `fullWidth` props for unique designs.
- Use `iconOnly` for buttons that only display an icon.
