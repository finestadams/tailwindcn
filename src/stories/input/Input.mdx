# Input

The `Input` component is used to capture user input. It supports different sizes, states, and configurations, making it a versatile component for forms and other user interfaces.

---

## Installation

To use the `Input` component in your project, after installing the package:
For configuration details, refer to the [Configuration Guide](?path=/docs/introduction--docs).

```bash
wind add input
```

---

## Usage

Import the `Input` component into your project:

```tsx
import { Input } from "@component/ui/input";
```

---

## Notes

- Use the `inputSize` prop to adjust the size of the input field.
- Combine the `error` prop with validation logic to display error messages.
- Use the `icon` prop to add icons for better user experience (e.g., search or email icons).
- The `disabled` prop can be used to disable the input field when necessary.
